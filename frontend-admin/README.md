# Sotatek Starter User Fronend 

## Setup

#### Step 1
```
npm i
```


#### Step 2
Create ``.env`` file
```
cp .env.example .env
```




#### Step 3: Run development server
```js
npm start
```

Default port running in ``9666``

You can change in ``.env`` file:
```
PORT=9666
```


#### Step 4

Access:
```
http://localhost:9666/#/
```

Enjoy and Finish !!!
